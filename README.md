## Packages
`brew install gh lazygit ripgrep fzf bat eza tldr`  
(Or any package manager)

gh — GitHub CLI. Interact with GitHub (PRs, issues, repos, workflows) from the terminal without touching a browser.  
lazygit — Terminal UI for git. Visual branch/commit/diff management without memorizing complex git commands.  
ripgrep (rg) — Extremely fast recursive text search. Replaces grep, respects .gitignore by default.  
fzf — Fuzzy finder. Pipe anything into it and get an interactive, filterable list. Pairs with shell history, file search, etc.  
bat — cat with syntax highlighting, line numbers, and git diff indicators in the gutter.  
eza — Modern ls replacement. Color-coded output, icons, git status integration, tree view.
